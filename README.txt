# 🚀 Drone Delivery Simulation (Java)

## Overview
A Java-based simulation system modeling a drone delivery network in a futuristic city. The system simulates logistics, drone behavior, and external events such as crashes and robberies.

## Simulation Details
The simulation models drone activity over time, including:
- Order assignment and prioritization
- Drone movement between locations
- State transitions (DOCKED, IN_TRANSIT, DELIVERING, COMBAT)
- Random events such as crashes and robberies
- End-of-day statistics and performance metrics

## Purpose
This project was developed as part of an Object-Oriented Programming course, with a focus on designing and managing systems involving multiple interacting objects. 

A key objective was to model and handle different types of drones, including delivery drones (armored and high-speed) and enemy drones (gatling and EMP), each with distinct behaviors and interactions within the simulation.

## Technologies
- Java
- Object-Oriented Programming (OOP)

## How to Run
1. Compile the project
2. Run `Main.java`
3. Ensure data files in the `/Data` folder are available

## Project Structure
- `Drones/` – drone classes and behavior
- `Data/` – input files
- `Simulation.java` – simulation logic
